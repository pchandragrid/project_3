Image Captioning Project

An advanced AI system that translates visual content into natural language descriptions using state-of-the-art Transformer models.

Executive Summary

This project implements an end-to-end Image Captioning System capable of automatically generating descriptive text for any given image. By leveraging and fine-tuning powerful vision-language models—BLIP, ViT-GPT2, and GIT—we achieve high-quality, human-like captions.

The system is designed not just for performance but for usability and experimentation. It includes a training pipeline for custom datasets (MS COCO), a comprehensive evaluation suite using industry-standard metrics like CIDEr, and a user-friendly Streamlit web application for real-time inference.

Computer Vision NLP Transformers PyTorch Streamlit

Key Features

Multi-Model Support

Compare results from three distinct architectures: BLIP (Bootstrapping Language-Image Pre-training), ViT-GPT2, and GIT (Generative Image-to-text Transformer).

Interactive Web App

A polished Streamlit interface allows users to upload images, tweak generation parameters (Beam Search, Max Length), and visualize attention maps.

Advanced Training

Custom training loops with gradient checkpointing, mixed-precision training (MPS/CUDA), and early stopping based on validation CIDEr scores.

In-Depth Analysis

Tools to analyze the impact of caption length, beam search size, and dataset quality on model performance.

Project Goals

  • Performance: Improve CIDEr scores by >10% over baseline pre-trained models through fine-tuning on COCO.
  • Usability: Democratize access to advanced AI models via a simple web interface.
  • Research: Understand the trade-offs between different transformer architectures for vision-language tasks.
