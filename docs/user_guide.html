User Guide

Learn how to use the Image Captioning Web Application to generate descriptions for your images.

Accessing the Application

The application is available as a web-based interface built with Streamlit. You can access it in two ways:

  1. Online (Hugging Face Spaces): Visit the hosted version at huggingface.co/spaces/pchandragrid/image_captioning.
  2. Locally: Follow the Developer Guide to run the app on your own machine.

Step-by-Step Instructions

1. Upload an Image

Click the "Browse files" button or drag and drop an image file (JPG, PNG, JPEG) into the upload area. The app supports standard image formats.

2. Select Models

In the sidebar (left panel), choose which AI models you want to use. You can select one or multiple models to compare their outputs side-by-side.

  • BLIP: Best overall performance.
  • ViT-GPT2: Faster but slightly less accurate.
  • GIT: Good for detailed descriptions.

3. Adjust Settings (Optional)

Fine-tune the generation process using the sidebar controls:

  • Beam Size: Controls the search width. Higher values (e.g., 5) generally produce better captions but take longer.
  • Max Length: The maximum number of words in the generated caption.

4. Generate Captions

Click the "Generate Captions" button. The app will process the image and display the results below.

Understanding the Controls

Beam Search vs. Greedy Search

By default, models might use "Greedy Search," picking the most likely next word at each step. Beam Search explores multiple potential sentence paths simultaneously.

  • Beam Size = 1: Greedy Search (fastest).
  • Beam Size > 1: Beam Search (better quality). We recommend a Beam Size of 3-5 for the best balance.

Max Length & Min Length

These settings constrain the output. If you want short, punchy captions, reduce the Max Length. For more detailed descriptions, increase it (though the model might start repeating itself if set too high).

Interpreting Results

Once generation is complete, you will see:

  • Generated Caption: The text description produced by the model.
  • Confidence Score (BLIP only): How confident the model is in its prediction.
  • Attention Map (BLIP only): A heatmap overlay showing which parts of the image the model focused on while generating specific words.
