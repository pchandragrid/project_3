Task 3 – Beam Search & Length Penalty Ablation

Runs a decoding grid over beam size and length penalty, evaluates caption quality (BLEU-4, METEOR, CIDEr, ROUGE-L), measures latency, and summarizes the quality-vs-speed trade-offs.

What this task does

Decoding grid

beam_size ∈ {1, 3, 5} and length_penalty ∈ {0.8, 1.0, 1.2}.

Quality metrics

Computes BLEU-4, METEOR, CIDEr, and ROUGE-L per configuration.

Latency metrics

Records mean latency and p95 latency (ms) to reflect performance trade-offs.

Visualization

Generates a CIDEr heatmap and a compact quality-vs-speed summary.

How to run

Full generation + evaluation

python -m tasks.task3_beam_ablation.run_beam_ablation \
  --checkpoint_dir tasks/task1_blip_optimization/checkpoints/blip_gc_mp_224 \
  --annotation_path src/data/raw/captions_validation.jsonl \
  --image_dir src/data/raw/val2017 \
  --num_samples 500 \
  --beam_sizes 1,3,5 \
  --length_penalties 0.8,1.0,1.2 \
  --device cpu

Recompute metrics/heatmap only

python -m tasks.task3_beam_ablation.run_beam_ablation \
  --num_samples 500 \
  --device cpu \
  --reuse_artifacts
