Project 3 – Task Execution & Results Report

A detailed, multi-page report documenting what was implemented across all five tasks, how to run each task, and where to find every important artifact and output figure.

Quick Start

These tasks are designed to be run as Python modules from the repository root.

Prerequisite: a Task 1 checkpoint

Tasks 2–5 expect a BLIP checkpoint produced by Task 1. In the READMEs, the canonical checkpoint directory is: 

tasks/task1_blip_optimization/checkpoints/blip_gc_mp_224

Recommended execution order

  1. Task 1: train/export/quantize/benchmark BLIP
  2. Task 2: attention visualization + grounding alignment
  3. Task 3: beam search + length penalty ablation (quality vs latency)
  4. Task 4: caption diversity + CAV-style steering
  5. Task 5: toxicity/bias audit + mitigation + fairness report
