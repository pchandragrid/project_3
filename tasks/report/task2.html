Task 2 – Attention Visualization and Cross-Attention Rollout

Generates captions while extracting per-step decoder cross-attention, renders attention grids, performs cross-layer rollout, and computes IoU-based grounding between attended words and COCO object boxes.

What this task does

1) Caption generation + attention extraction

Runs the checkpoint model and extracts cross-attention at each decoding step.

2) Per-step attention maps

Renders token-step heatmaps that show “where the model looked” while generating each word.

3) Cross-layer rollout

Creates rollout maps to aggregate attention across layers for more interpretable grounding patterns.

4) Grounding with IoU

Uses COCO object boxes to compute mean alignment IoU (word grounding quality).

Folder layout

tasks/task2_attention_analysis/
  run_attention_analysis.py
  attention_utils.py
  artifacts/
    attention_maps/   # .npz per sample/step/layer
    rollout_maps/     # .npy per sample/step
  results/
    figures/         # attention_grid_XXXX.png
    reports/         # CSV + markdown summaries

How to run

python -m tasks.task2_attention_analysis.run_attention_analysis \
  --checkpoint_dir tasks/task1_blip_optimization/checkpoints/blip_gc_mp_224 \
  --annotation_path src/data/raw/captions_validation.jsonl \
  --image_dir src/data/raw/val2017 \
  --instances_json src/data/raw/instances_val2017.json \
  --num_samples 100 \
  --max_new_tokens 20 \
  --grid_steps 5 \
  --device cpu

Key dependency

This task expects Task 1’s BLIP checkpoint output. If you have a different checkpoint directory, update --checkpoint_dir accordingly.

Key outputs

  • tasks/task2_attention_analysis/results/reports/attention_summary.md
  • tasks/task2_attention_analysis/results/reports/alignment_details.csv
  • tasks/task2_attention_analysis/results/figures/attention_grid_XXXX.png (100 grids)
  • tasks/task2_attention_analysis/artifacts/attention_maps/ and rollout_maps/

Interpretation

  • Higher mean IoU implies better grounding between attended decoding steps and object regions.
  • Very long captions can show lower mean IoU due to attention diffusion.
  • Inspect low-IoU words for possible hallucination candidates.
